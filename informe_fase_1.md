## ✅ Tareas Completadas

### 1. **Infraestructura del Proyecto**
- Dominio y hosting contratados con SiteGround.
- Web básica de NucleoIA-labs creada y operativa.
- Cuenta de correo profesional `admin@nucleoia-labs.com` creada y conectada.

### 2. **GitHub**
- Repositorio central creado.
- Estructura del repositorio organizada:
  - `docs/` para documentación
  - `docs/progreso/` para informes de avance
- Primeros archivos README y roadmap subidos.

### 3. **Google Calendar**
- Calendario de ejecución inicial creado con tareas generales.
- Revisión de planificación: sábados y domingos ajustados como días de tareas menores.

### 4. **Gestión Financiera**
- Google Sheets creado para control de gastos del proyecto.
- Incluye: 
  - Coste anual de SiteGround + dominio (93,87€)
  - Suscripción mensual a ChatGPT Plus

### 5. **Redes Sociales**
- Creadas cuentas en Instagram, TikTok, YouTube.
- Imagen de perfil unificada.
- Primer post diseñado.
- Estrategia futura: Automatización de publicaciones.

### 6. **Automatización**
- Cuenta de Make conectada.
- Primera automatización operativa:
  - Detecta nuevas filas en Google Sheets.
  - Filtra por estado “Pendiente”.
  - Envía correo automatizado con formato HTML.
  - Actualiza estado a “Programado”.
